# 📁 Сохранение — снимки по каждому шагу

По одному файлу на шаг (M1, M2, …). Каждый файл — «заморожённый» снимок: что и как
сделано, нюансы, как проверить, на чём остановились.

## Правила
- **Один шаг = один файл.** После шага создаётся новый и больше НЕ редактируется.
- **Текущее состояние тут НЕ хранится** — оно в `../НАЧНИ-ОТСЮДА.md` (§8). Снимки заморожены,
  чтобы не возникало устаревших копий.

## Как пользоваться
- **Новый чат:** читать `../НАЧНИ-ОТСЮДА.md` (там ссылка на последний снимок). Все подряд читать не нужно.
- **Понять шаг / откатиться:** открыть нужный файл (код версии = git-тег вверху).
