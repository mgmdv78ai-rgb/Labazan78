# M1 — Установка системы сохранения

- **Код версии (git):** M1 (тег)
- **Дата:** 2026-06-22

## Что сделано (простым языком)
Поставили систему сохранения проекта: теперь после каждого шага можно делать «чекпоинт» —
проект сохраняется в git и улетает в GitHub. Любой новый чат сможет понять проект по одному
файлу `НАЧНИ-ОТСЮДА.md` и продолжить с того места, где остановились.

## Как сделано (для ИИ)
- Скил `save-system` и команда `/save` распакованы в `~/.claude/` (только на эту сессию).
- В репозиторий добавлены: `НАЧНИ-ОТСЮДА.md`, `ИСТОРИЯ-ВЕРСИЙ.md`, `ВОССТАНОВЛЕНИЕ.md`,
  `.gitignore`, `.save-system-version`, `scripts/backup-to-drive.sh`, `Сохранение/`.
- Адаптация под облачный Claude Code (Linux): облачная копия = GitHub; Google Диск-бэкап
  отключён (скрипт оставлен на будущее для Mac, в облаке корректно пропускается).
- Тип проекта: статический сайт (есть `index.html`) → `.gitignore` минимальный.

## Нюансы, открытые на шаге
- `~/.claude/` в облаке стирается между запусками → скил и `/save` придётся переустановить из zip.
- Снимок создан ДО `git tag`, в «Код версии» указан заранее известный тег `M1` (см. SKILL.md).

## Как проверить
- `cat НАЧНИ-ОТСЮДА.md` — онбординг на месте, §8 актуален.
- `git tag` — есть `M1`; `git log --oneline` — есть коммит установки.
- `bash scripts/backup-to-drive.sh M1` — корректно сообщает, что Google Диск не найден, и выходит.
